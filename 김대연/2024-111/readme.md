# Approach 
- 큐에 모든 익은 토마토를 넣는다.
- 큐에서 하나씩 꺼내면서 상하좌우를 확인하며 이동한다
- 이동할 때마다 distance를 1씩 증가시킨다.
- 이미 방문한 곳은 방문하지 않는다.(distance가 -1이 아닌 곳)
- 모든 토마토가 익었는지 확인한다.
- 모든 토마토가 익었다면 distance를 return한다.
- 모든 토마토가 익지 않았다면 -1을 return한다.
